
# Portfolio

This is the source code used to introduce myself.

## Install

Clone the project

```bash
  git clone https://github.com/mkhue720/portfolio
```

Go to the project directory

```bash
  cd portfolio
```

Install dependencies

```bash
  npm install
```
Start

```bash
  npm run dev
```


## Deployment

To deploy this project run

```bash
  npm run build
```
